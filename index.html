    TimeAhead

    "Real world Business Research Analyst Use Case of Amazon. Implemented End To End Machine Learning Model with Deployment"

    I successfully addressed a use case for Amazon Business Research Analyst by accurately predicting order delivery times. Throughout the process, I established three distinct environments: research, development, and production. In the research environment, I collected data from Kaggle and meticulously converted the data from unstructured form into structured form, conducted comprehensive exploratory data analysis to gather some of the information regarding the dataset, implemented advanced feature engineering techniques to deal with all the problems analyzed in EDA like dealing with missing values converting categories into numeric values etc., carefully selected relevant features using lasso technique, and diligently constructed and optimized a predictive model using GridSearchCV. Transitioning into the development environment, I transformed the code into a modular structure and expertly developed a web application using Flask and my web development expertise, In such a way whenever user will click on the sumbit button then in the backend data will interact with pickles files that we have created and predictions will take place. Finally, In the production environment, I seamlessly deployed the model using Amazon Elastic Beanstalk and skillfully integrated it with a code pipeline on GitHub to ensure continuous delivery.

    FashionFinderMyntra

    "I designed and implemented an advanced Content based recommendation system using machine learning for Myntra's women's fashion products. The system not only provides personalized recommendations but also offers an interactive user interface for a seamless browsing experience."

    In the Research environment, I gathered a Kaggle dataset and performed essential data preprocessing tasks, including handling missing values, removing duplicates, and eliminating unnecessary HTML elements using regular expressions and Python. I created a streamlined dataframe containing relevant features for my recommendation system. To enhance text analysis, I utilized the NLTK library for stemming and the CountVectorizer from scikit-learn to determine word frequencies. Using cosine similarity, I measured document similarity, a crucial aspect of our recommendation system. The recommendation function was implemented, and pickle files were saved for future use in the web application. Moving to the Development environment, I designed a visually appealing and user-friendly web application using Flask as the backend framework. Employing HTML, CSS, Bootstrap, and JavaScript, I created an interactive UI inspired by Myntra's official website. GitHub was utilized for collaboration, with best practices followed for repository setup. A clean and isolated development environment was established using Anaconda, ensuring efficiency. Various files such as setup.py, requirements.txt, and directories like templates and static were created to construct a dynamic interface. Proper version control was ensured by utilizing .gitignore to exclude unnecessary files. Overall, my work resulted in an immersive web application with an interactive UI resembling Myntra's website. The project's well-organized structure, collaborative repository, and efficient development process utilizing a virtual environment contribute to its success.
